#Semesterarbeit PROG2: FILMZER

Idee:
Wenn man keinen guten Film auf Netflix (oder wo auch immer) findet um zu schauen, wird man schnell ungeduldig. Deshalb dachte ich mir "Filmzer" aus. Ein Programm, welches dem User fragen stellt, um ihm dann einen vorgeschlagenen Film auszuspucken. Dem User werden Fragen zu seiner aktuellen Stimmung gestellt. Auf dieser Art werden weitere Fragen gestellt. Anhand von den Eingaben werden dann die Antworten mit meiner Film liste abgeglichen.
Diese Liste wurde in einem json File eingetragen. Beispielsweise ist pro Film eine Zahl von 1 bis 10 zu "lustig" angegeben (und auch zu den weiteren Stimmungen.) So soll mein Programm dann die Eingaben von User zusammentragen und den passenden Film, oder mehrere, ausspucken.
